Projects

  • NaviAI | Python, Django, PostgreSQL, JavaScript, Tailwind CSS, AWS, GPT-4o

    Senior Capstone Project, University of Utah, built in Spring 2025

    NaviAI is a Dungeons and Dragons (D&D) web application made with the express purpose of aiding newer players with learning how to play the game for the first time. The application utilizes the powerful language model API, GPT-4o, to analyze information about the game and user input to provide advice to players on how to proceed in the game. The goal of the application is to provide new players a way to help them make D&D less daunting to approach, and to provide experienced players with a powerful resource to support and enhance their D&D experience.

    My roles in the project consisted of the following:

    • Supervised a team of 4 software developers by efficiently managing project timelines and development tasks using agile methodologies and project management tools.
    • Led the development of a Django-based web application integrating GPT-4o to assist players with Dungeons & Dragons gameplay, and optimized a PostgreSQL database to manage game-related data efficiently.
    • Spearheaded the development of a functional chat between users using web sockets, and the development of a user-friendly UI/UX design to provide real-time, context-aware responses from the AI, as well as the ability to interact and play D&D with other players.
    This project was designed by my senior capstone team in Fall 2024, and we developed the project over the course of Spring 2025.
